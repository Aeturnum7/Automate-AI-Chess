# Automate-AI-Chess
An automated Ai chess board using Python


## parts used
 - Motors Nema 17
 - Electromagnet
 - L298 H-brdge
 

## TO RUN THE CODE 

`python3 src/main.py`

### TEST Files

`Manual.py` Moves the Motor Manually, giving it inital and fina coordinates
`emtest.py` Let's you controll the electromagnet
